# **App Name**: Your Buddy

## Core Features:

- Student Registration: Allow students to register and create profiles using Firebase Authentication.
- Profile Creation: Enable students to list their interests, skills, and upload profile pictures.
- Student Discovery: Allow students to search and discover other students based on interests and skills using Firestore.
- Connection Recommendations: Suggest connections based on shared interests using AI to filter appropriate student connections. The tool will help deciding which students to display.
- Team Formation: Facilitate the creation of teams for college projects and manage team members using Firestore.
- Chat Feature: Enable real-time chat between students and teams using Firestore.
- UI with Tabs: Implement a user interface with tabs for Discover, Connections, Teams, and Profile.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and tech-focused feel.
- Background color: Light blue-gray (#E8EAF6), a very desaturated variant of the primary color.
- Accent color: Purple (#7E57C2), analogous to the primary, for highlighting interactive elements.
- Body and headline font: 'Inter', a sans-serif with a modern, neutral look, suitable for both headlines and body text.
- Use clean, outlined icons from a library like Material Icons for a modern UI.
- Implement a tab-based layout for easy navigation between Discover, Connections, Teams, and Profile.
- Subtle animations for transitions and loading states to enhance user experience.
